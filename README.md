
# Translational Oncology Analytics Using MSK-IMPACT Data

## Project Overview

This project explores clinical and genomic patterns in breast cancer cohorts using real-world oncology sequencing data from the MSK-IMPACT study available through cBioPortal.

The analysis combines SQL, Python, and Power BI to simulate a translational oncology analytics workflow focused on clinical cohort design, genomic data integration, mutation landscape analysis, and healthcare-oriented reporting.

The project was developed as a portfolio case study oriented toward healthcare data analytics, clinical research data workflows, and translational oncology applications.

---

# Objectives

- Build a relational clinical-genomic database using real-world oncology data
- Design clinically relevant breast cancer cohorts
- Explore mutation patterns across molecular subtypes
- Integrate clinical and genomic variables for exploratory analysis
- Develop interactive dashboards for clinical and translational insights

# Dataset

Source:
MSK-IMPACT Clinical Sequencing Cohort (cBioPortal)

The project focuses on breast cancer subsets from the MSK-IMPACT real-world oncology dataset, including:

- Clinical patient data
- Clinical sample data
- Somatic mutation annotations
- Cohort-level genomic information
